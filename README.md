# Visual Algorithm Simulator

Visual Algorithm Simulator is a modern desktop application for interactive visualization, analysis, and learning of computer science algorithms.

## Features

- 🎨 Modern desktop UI built with PySide6
- ⚡ Smooth animations with step-by-step execution
- 📊 Real-time statistics
- 📖 Live explanation panel
- 🧩 Plugin-ready architecture
- 🌙 Theme system
- ⏯ Pause, resume, step, speed control
- 📹 Planned GIF/MP4 export

## Supported Categories

- Sorting Algorithms
- Searching Algorithms
- Graph Algorithms
- Tree Algorithms
- Dynamic Programming
- Computational Geometry

## Tech Stack

- Python 3.13+
- PySide6
- NumPy
- NetworkX
- PyQtGraph
- Loguru

## Project Structure

```
core/          Application engine
algorithms/    Algorithm implementations
graphics/      Rendering and animations
models/        Data models
operations/    Visualization operations
ui/            User interface
assets/        Icons, themes, fonts
```

## Installation

```bash
git clone https://github.com/Mluffy7456/VisualAlgorithmSimulator.git
cd VisualAlgorithmSimulator
pip install -r requirements.txt
python main.py
```
